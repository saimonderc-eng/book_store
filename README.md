*Book Store Api*

REST API для книжного магазина с поддержкой аутентификации, ролей пользователей, корзины и заказов.
Проект реализован на Spring Boot с использованием JWT-аутентификации, Spring Security, JPA Hibernate и PostgreSQL.

СТЕК технологий:

	- Java 17
    - Spring Boot
    - Spring Security
    - JWT
	- liquibase
    - Spring Data JPA
    - Hibernate
    - PostgreSQL
    - MapStruct
    - Lombok
	-docker



*Функционал*


//

Аутентификация и пользователи:

 • Регистрация и логин пользователей.

 • Изменение пароля.

 • JWT аутентификация.

 • Роли(Admin, Manager, User), привязывание ролей(Admin).

 • Разграничивание доступа через @PreAuthorize

 • Блокировка пользователей(Admin).

 • Изменение данных пользователя.


//

Книги:

 • Получение списка книг.

 • Добавление / удаление / удаление книг(Admin, Manager).

 • Поиск книг по:

   · категориям

   · ключевым словам

• Soft  delete

//

Категории:

 • Привязывание книг к нескольким категориям (Many-To-Many).

//

Корзина и заказы:

 • Добавление товара в корзину

 • Получение списка товаров в корзине

 • Оформление заказа из корзины

 • Создание заказа(только для manager, admin)
